This guides you to setting up your computer to run all my scaffolds.

Though this guide is for the Ubuntu linux; it should be easy to retrofit to others.

Though this guide has been tested on macOS High Sierra; it will probably work on older versions.

Follow this guide on Windows 7+.

All grey background assumes you are running in an active Command Prompt (cmd.exe) Terminal session

Step 0: Install build dependencies

sudo apt update sudo apt install build-essential git-core tcl Latest XCode and Command Line Tools /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" brew update brew install git
Ensure you check "Add to PATH" whenever asked:
  1. Build Tools for Visual Studio 2017
  2. git

Test that it installed correctly with:

git --version cl /?

Step 1: Install Node.js

curl -L https://git.io/n-install | bash -s -- -y lts

Download and setup from latest: 'Windows Installer (.msi)' [LTS].

Test that it installed correctly with:

node --version npm --version

Step 2: Install Redis

# Alternatively build from source. Quick guide: https://askubuntu.com/a/868862 sudo add-apt-repository ppa:chris-lea/redis-server sudo apt update sudo apt install redis-server brew install redis brew services start redis

Download and install the latest .msi.

You can test it's installed and running with:

redis-cli ping

If that failed, open a new session and run this, then try the ping again:

redis-server

Step 3: Install and configure Postgres

Follow their official guide. I prefer using latest version from their apt repository.

Follow their official guide. I prefer the EnterpriseDB package.

Run this in an Administrator Command Prompt (how-to):

POSTGRES_DB_USER='REPLACE_ME' REST_PASS='REPLACE_ME' # recommend using `read -s REST_PASS` POSTGRES_DB_NAME='REPLACE_ME' createuser --superuser "$POSTGRES_DB_USER" psql -c "CREATE USER $POSTGRES_DB_USER WITH LOGIN PASSWORD '$REST_PASS';" createdb "$POSTGRES_DB_NAME" --owner "$POSTGRES_DB_USER" export RDBMS_URI="postgres://$POSTGRES_DB_USER:$REST_PASS@localhost/$POSTGRES_DB_NAME" # ^Recommend adding this `export` line to your ~/.bash_profile set POSTGRES_DB_USER="REPLACE_ME" set REST_PASS="REPLACE_ME" set POSTGRES_DB_NAME="REPLACE_ME" createuser --superuser "%POSTGRES_DB_USER%" psql -c "CREATE USER %POSTGRES_DB_USER% WITH LOGIN PASSWORD '%REST_PASS%';" createdb "%POSTGRES_DB_NAME%" --owner "%POSTGRES_DB_USER%" setx RDBMS_URI "postgres://%POSTGRES_DB_USER%:%REST_PASS%@localhost/%POSTGRES_DB_NAME%" :: This last line will add RDBMS_URI to your environment

Alternatively use my script.

You can test it's installed and running with:

psql "$RDBMS_URI" -c 'SELECT 1' psql "%RDBMS_URI%" -c "SELECT 1"
